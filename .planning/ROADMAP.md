# Roadmap: GDPdU DuckDB Extension

## Overview

DuckDB C++ extension that reads German tax audit (GDPdU) exports with a single function call. v1.0 delivered local import. v1.1 added Nextcloud batch import via WebDAV.

- ✅ **v1.0 Local Import** — Phases 1-5 (shipped 2026-01-22)
- ✅ **v1.1 Nextcloud Batch Import** — Phases 6-8 (shipped 2026-02-13)
